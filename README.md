## 📋 Requirements

Before starting, make sure you have:

- Node.js 18+
- npm
- Gemini API Key

---

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/TheRynzo.git
cd TheRynzo
```

Install dependencies:

```bash
npm install
```

---

## 🔑 Environment Setup

Create a file named:

```env
.env.local
```

Add:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## 🚀 Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

## 🏗 Build Production

```bash
npm run build
```

Start Production:

```bash
npm start
```
